# Dynamic Pathfinding Agent

This project implements **A* (A-Star)** and **GBFS (Greedy Best-First Search)** algorithms with a **Tkinter GUI**.

## How to Run

1. Make sure you have **Python 3.11+** installed.
2. Open terminal / command prompt.
3. Navigate to the project folder:

cd "E:\4th sem\AI\Assignment#2"

4. Run the program:

python Assignment#2_Q#6.py

5. The GUI allows you to:
   - Place start and goal
   - Add/remove walls
   - Select algorithm (A* / GBFS)
   - Select heuristic (Manhattan / Euclidean)
   - Click **Run Search** to find the path
   - Enable **Dynamic Obstacles** to see the agent move automatically

## Notes

- No external libraries are required (Tkinter comes with Python)
- You can test different scenarios using the GUI
